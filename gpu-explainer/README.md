# how our words become images

an interactive explainer mapping the full pipeline from text prompt to pixel through flux dev, pytorch, and the rtx 5090.

**three depth levels:**
- plain english ... for anyone curious
- technical ... for builders
- on the 5090 ... for hardware heads

**built by og**

## deploy

```bash
npm install
npm run dev     # local
npm run build   # production
```

deploy to vercel: connect this repo and it auto-detects vite.

## note

replace `public/og-image.png` with a 1200x630 screenshot for twitter card preview.
